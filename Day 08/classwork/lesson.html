HTML

HTML (HyperText Markup Language) გამოიყენება ვებ გვერდების სტრუქტურის განსაზღვრისთვის.

CSS

CSS (Cascading Style Sheets) გამოიყენება ვებ გვერდების დიზაინისა და ვიზუალური სტილის განსაზღვრისთვის.

JavaScript

JavaScript გამოიყენება ვებ გვერდების ინტერაქტიულობისა და დინამიური ფუნქციონირების დასამატებლად.
